Welcome to StinkyPete

An IE7-based web browser for viewing aging, cantankerous web pages and OnBase E-Forms up until version 15, which now uses an IE9-based rendering engine.

This browser's User Agent should match the User Agent of your OnBase installation.

Readme.md

The readme.md file in the solution root describes how to create a custom start page, develop with all resources local, and deploy to OnBase, among other morsels.
